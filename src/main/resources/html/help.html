Time Table Creator


Home

This is the main screen for a user which is shown after a succesfull login.

New

Used to create a new Time Table

Logout

Logs out the current user and navigates to log in screen

Tile Pane - Gray area

Lists all the time tables that have been created previously. Click on any one to open.

Subjects

Take the user to Subject window where the user can view subject list, add new subject or delete an existing subject.

Teachers

Take the user to Teachers window where the user can view teacher list, add new teacher or delete an existing teacher. The "ViewTimeTable" button on the Teachers page shows the time table for that particular teacher. It shows the subject and timetable name corresponding to weekday and time period.

Users

Take the user to Users window where the user can view user list, add new user or delete an existing user.


Edit Periods

On this screen you change the default values of the periods from 1,2.....9,10 to any custom calue which will reflect in the time table. You can change for any one period or all the periods.

Reset

Resets the value for all the periods to default value

Save

After editing the values, you have to click on Save button to make the changes permanent.


Timetable

Edit

Enables edit mode for the timetable wherein you can change the subject and teacher name for all the cells in the timetable.

Save

Save button is disabled by default. It gets enabled once user enable Edit mode. Once you have made the changes to the timetable, you can click on Save to make the changes permanent.

Cancel

Cancel button is disabled by default. It gets enabled once user enable Edit mode. While in the Edit mode, If a user is not satisfied with the changes, he can click on Cancel to exit the Edit mode without saving the changes.

Export

This button is used to create an Excel file of the current timetable and save it to the local hard drive. User can also selcet to open google drive tab in browser to backup the excel file to cloud.

Delete

This button deletes the timetable from the system. All the data of the timetable will be deleted completely.
